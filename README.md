# Markdown Writer

一款专业的Markdown写作工具，基于Vue 3 + Vite开发，提供实时预览、图片上传、样式复制等功能。

## 功能特点

### 核心功能
- **Markdown编辑器**：支持标准Markdown语法，语法高亮，快捷键支持，自动保存功能
- **实时预览**：分屏预览，实时同步，支持代码块高亮，支持表格渲染，支持数学公式渲染
- **图片上传**：支持拖拽上传，支持剪贴板粘贴上传，自动生成Markdown图片链接，上传进度显示
- **样式复制**：支持复制为纯文本，支持复制为HTML格式（带样式），支持复制为Markdown格式
- **本地存储**：自动保存到localStorage，支持手动保存，支持文章管理（新建、删除、重命名）

### 辅助功能
- **主题切换**：支持浅色/深色主题，支持自定义主题
- **导出功能**：支持导出为Markdown文件，支持导出为HTML文件
- **快捷键**：常用Markdown语法快捷键，编辑器操作快捷键

## 技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **Markdown解析**：marked.js
- **代码高亮**：highlight.js
- **图片上传**：SM.MS API

## 使用指南

### 创建新文章
1. 在首页点击「新建文章」按钮
2. 输入文章标题
3. 在编辑器中编写Markdown内容
4. 系统会自动保存，也可以点击「保存」按钮手动保存

### 编辑现有文章
1. 在首页点击文章卡片进入编辑页面
2. 修改文章内容
3. 系统会自动保存

### 导出文章
1. 在编辑页面点击「导出Markdown」按钮导出为.md文件
2. 点击「导出HTML」按钮导出为.html文件

### 图片上传
1. 在编辑器中拖拽图片到编辑区域
2. 或使用剪贴板粘贴图片
3. 图片会自动上传到图床，并生成Markdown图片链接

### 主题切换
1. 点击顶部导航栏的「设置」按钮
2. 在设置页面选择「浅色主题」或「深色主题」

## 注意事项

- 文章保存在浏览器的localStorage中，清除浏览器数据会导致文章丢失
- 建议定期导出重要文章
- 图片上传使用免费图床，可能有上传限制
- 支持主流现代浏览器（Chrome, Firefox, Safari, Edge）
